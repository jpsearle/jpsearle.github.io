The properties of the angular momentum operators and their eigenvalues can be derived from two considerations.

  1. Angular momentum is a vector, \( \hat{\boldsymbol{L}} \), and so its length squared \( \hat{L}^{2} \) is a scalar. \[ \hat{\boldsymbol{L}} = \left(\hat{L}_{x},\hat{L}_{y},\hat{L}_{z}\right) \] \[ \hat{L}^{2} = \hat{L}^{2}_{x} + \hat{L}^{2}_{y} + \hat{L}^{2}_{z} \]

  2. The commutation relation is defined as following, with a vector formulation (1) following from it. \[ \left[ \hat{L}_{a} , \hat{L}_{b} \right] = i \hbar \varepsilon_{abc} \hat{L}_{c} \] \[ \hat{\boldsymbol{L}}\times\hat{\boldsymbol{L}} = i \hbar \hat{\boldsymbol{L}} \tag{1} \] Where \(a\), \(b\) and \(c\) are the components \(x\), \(y\) and \(z\). Note that, whilst each component is Hermitian, the commutator is not, so there needs to be an imaginary component on the right side (since \(i^{∗}=−i\)).

Unlike two components, the squared length commutes with all components (2). Therefore, only the squared length and a single component can be observed simultaneously.
